/ Automation
/
/ Entities that can walk, or run, must be required to have movements for the game
/ to feel lively.

/ _: Whitespaces.
/ @: Trigger name.
/ ^: [Direction, Steps]. Can be chained for delaying scripted movements.
/ $: Start of script.
/ %: Script delimiter.
/ #: Speech Dialogue.
/ /: Comments. Gets ignored.
/ ?: Question Dialogue.
/ +: Affirmative dialogue. 
/ -: Negative dialogue
/ [: Affirmative Action
/ ]: Negative Action

/ DO NOT CHANGE.
$0
@Eraser
%

@Testing_Script

#Hello_world.
^D3U3R1L0
%

$2
@Clockwise_Circle
?Going to run in a circle?
+R2D2L2U2

-R2D2L2U2
